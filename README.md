✈️ 自分だけのフライトマップ
 旅の記録を地図に刻む、あなただけのフライトログ


📖 どんなアプリか
旅行の記録を登録すると、地図上で訪れた空港同士が線で結ばれるWebアプリです。
「今まで自分はどこに行ったんだろう？」が一目で分かり、写真を追加することで当時の思い出も一緒に振り返ることができます。


🌱 なぜ作ったか
私は、「思い出」というものをとても大切にしています。
これまで海外へ足を運び、そこで感じた空気や新しい発見を、かけがえのない「思い出」としていつでも鮮明に振り返れるようにしておきたいからです。
そんな中ふと、「自分が世界をどう飛び回ってきたのか、その『移動の軌跡』そのものも、一つの思い出なのではないか？」そう考えるようになりました。
もちろん、既存の旅行アプリなどでも記録をつけることはできます。
でも、私にとって特別な意味を持つ旅の記憶だからこそ、もっと自由で、もっと愛着の湧く形に残したい。
与えられたツールを使うだけでなく、「自分自身で思い出を詰め込む記録」というものを作りたいと思ったからです。


✨ 主な機能
　🗺️ フライトマップ表示 — 訪れた空港同士を線で結び、旅の軌跡を地図上に可視化
　📝 旅行記録の登録 — 出発地・目的地・旅行日・便名などを登録
